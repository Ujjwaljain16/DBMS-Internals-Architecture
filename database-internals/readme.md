# Database Internals – Learning Notes

This repository contains learning notes based on *Database Internals* by Alex Petrov.  
The notes are compiled through a combination of:

- book study,
- external references,
- personal understanding,
- manual research,
- and LLM-assisted explanations.

The goal of these notes is to make database internals easier to understand through structured explanations, simplified intuition, and consolidated learning material.
